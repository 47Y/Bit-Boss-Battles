Take charge, brave streamer, and lead your viewers into battle with their bits! The almighty Bit Boss has awoken! Give your viewers a live game to play against each other by using this widget to fight the Bit Boss. For each bit cheered, 1 damage is dealt to the current boss. But be forewarned; the viewer to deal the final blow to the current boss becomes the new Bit Boss!

Before you can use the widget, you must authorize it. Click the "Authorize" button to open the Auth window. Once you've finished, the Auth window will close and you can then click Launch. The demo will be around whether or not you authorize.

Click "Launch Widget" to launch the live widget in your current browser. If Bit Boss Battles won't work properly in your streaming software as a web source, this is an alternate solution. If you'd like to try Bit Boss Battles first, you can click "Launch Demo" to see how it works. NOTE: The demo ignores all settings except sound.

Click "Reset Cookies" to clear the widget's cookies. This clears the current boss, their health, and all options enabled.

Based on Overkill. Multiplier:
Constant Value:

Determines how much HP each Bit Boss gets. You can set a constant value, or use the spare bits left over from the final blow (the "overkill"), multiplied by the set multiplier. If Overkill is selected, the initial boss will have 1,000 HP. As an example for Overkill, the last attacker hits a boss at 50 remaining HP with 100 bits. The "overkill amount" will be 50. If the multiplier is set to 5, the new boss's HP will be set to 250. If the overkill amount with the multiplier is less than 100, the next boss's HP will be set to 100. NOTE: These settings require a manual restart.
